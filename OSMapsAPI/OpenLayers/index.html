OS Maps API demonstrator

This sample shows how to display a map using the OS Maps API and OpenLayers

To view the map, please enter a valid API key.

To connect to your mapping API:

  1. Go to the OS DataHub,
  2. Create a project,
  3. Add the OS Maps API (WMTS) API to your project
  4. Copy the project API Key, and enter it into the box above.
Note that this sample will allow you to zoom the map down into Premium data layers. If your API key only has access to OpenData layers then those requests will fail.
